## Como probar los tests

Ejecutamos el archivo **democracia.pl** con SWI-Prolog en una terminal de la siguiente manera

```bash

swipl democracia.pl

```
Y luego debemos ejecutar dos comandos
```prolog

load_test_files([]).

run_tests.

```
